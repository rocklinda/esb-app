# _CRUD Invoice_

## Installation

### Setup
```
composer install
```

### .Env
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=esb_app
DB_USERNAME=root
DB_PASSWORD=
```
### Migrate Database
```
php artisan migrate:fresh --seed
```

### Run Application
```
php artisan serve
```

### URL CRUD Invoice
```
http://127.0.0.1:8000/invoices
```

### API Get All Data
```
http://127.0.0.1:8000/api/invoices
```

### API Get One Data
```
http://127.0.0.1:8000/api/invoices/{id}
```
